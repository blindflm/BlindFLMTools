Blind FLM Operators

Blind FLM consists of six operators that are extended from the original KLM and the Fingerstroke-Level Model (FLM). Blind FLM consists of six operators, there are:

Tap
A user who is visually impaired would tap anywhere on the screen to select and listen to the audio description of the underlying element. This element can be a button, content, link, image, or vidoe.
Double Tap
While the tap operator is often used to select an element, double tap is required to activate the selected element. Therefore, a unique double tap operations is included.
Drag
A user who is visually impaired would slide three fingers to scroll up and down on the screen.
Flick
A user who is visually impaired woulf flick with one finger to navigate across an application's elements.
Mental Act
Users who are visullay impared rely on listening to conceptualize the real world and therefore their interaction with smartphones. This process is combined with the cognitive processing required to perform an action.
System Response
The time it takes the system to repond to an interaction.

A model is constructed using Blind FLM's operators, which can then be used to nemurically predict execution time using the following equation:

Texecute = TTap + TDouble Tap + TDrag + TFlick + TMental Act + TSystem Response
